# Quickstart: 검증 시나리오

구현 후 아래로 수용 기준을 확인한다. 앱 구동은 `run`/`verify` 스킬 사용.

## 사전

- 변경 파일: `npx tsc --noEmit` · `npx eslint` 클린.
- 지수 지표 검증 전: `krx_index_stats_cache` 행 확인. 비었으면 `npm run sync:krx-index`(최초엔 `$env:KRX_INTERACTIVE="1"`로 KRX 로그인) 1회 실행.

## US1 — 로고 (P1)

1. 보유 목록·검색에서 대표 4종 확인:
   - 삼성전자(005930) → **회사 로고**(favicon).
   - KODEX 200(069500) → **운용사 로고**(없으면 운용사 약칭 폴백).
   - 코스피(^KS11) 관심목록 → **🇰🇷 국기 이미지**.
   - 비트코인(BTC-USD) → **코인 이미지**.
2. 로고 미보유 종목(중소형주) → **글자 동그라미 폴백**, 깨진 이미지 0건.
3. 같은 종목을 다른 화면(상세·배분·수익률)에서 → **동일 이미지**(일관).

→ SC-001, SC-002 충족.

## US2 — 지수 지표 (P2)

1. 코스피 상세(`/index/^KS11`):
   - PER·PBR·배당수익률이 **값**으로 표시(캐시 충전 시).
   - **Forward PER 셀이 보이지 않음**(국내·미국 모두).
   - 캐시 미충전 상태면 해당 셀 "**데이터 준비 중**"(빈칸 아님), 충전 후 값.
2. 미국 지수(`/index/^GSPC`): 출처 없는 지표는 "**정보 없음**", Forward PER 없음.

→ SC-003, SC-004 충족.

## US3 — 환율 상세 (P3)

1. `/cash?tab=fx` 환율 목록에서 **USD 행 탭** → `/fx/USD` 진입(체감 즉시).
2. 상세에 **현재 환율 + 일간 변동 + 추이 차트 + 52주 고저** 표시.
3. **뒤로가기** → `/cash?tab=fx` 복귀.
4. 차트 소스 실패해도 **현재 환율 유지**, 차트 자리 대체 문구.
5. 레이아웃(탭바·뒤로가기·헤더)이 **종목·지수 상세와 동일**.

→ SC-005, SC-006 충족.

## 회귀

- 기존 아바타 사용처 8 페이지 + 컴포넌트 렌더 정상(빈 화면·콘솔 에러 없음).
- 지수 상세의 차트·섹터·구성종목·버핏지표 기존대로.
- 원장·보유 수량·평가 불변(읽기 전용 변경).
