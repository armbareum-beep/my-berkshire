# UI Contract: 간편모드 UI 업그레이드

외부 API가 없는 프론트엔드 기능이므로, 계약은 **사용자에게 노출되는 시각·상호작용 약속**으로 기술한다.

## C1 — 헤더 워드마크 (US1 / FR-001~003)

**위치**: 대시보드 홈(`/dashboard`) 헤더 좌상단 영역.

**시각 계약**:
- "ENUF" 글자 워드마크가 헤더 좌상단 최상단에 표시된다(별도 심볼/아이콘·배경칩·그라데이션 없음).
- 워드마크: 굵은 잉크색 타이포(`--foreground`), 회사명보다 시각적으로 상위.
- 그 아래에 기존 회사명(`{holding.name} ›`, /company 링크)·설립일·모드가 하위 정보로 유지된다.

**상호작용 계약**:
- 워드마크 자체는 클릭 대상이 아니어도 된다(브랜드 표식). 회사명 링크의 기존 동작(/company 이동)은 보존된다.

**불변(회귀 금지)**:
- 우측 액션(검색 버튼, 로그아웃)의 위치·동작 불변.
- 폭 360~480px에서 좌측 텍스트 블록과 우측 액션이 겹치지 않는다.

## C2 — 보유 행 진입 화살표 (US2 / FR-004~006)

**위치**: 보유종목 목록의 각 종목(잎) 행 — 계좌별 모드(`AccountGroups`)와 전체 종목 모드(`HoldingsBrowser` flat) 양쪽.

**시각 계약**:
- 각 진입 가능 행의 우측 끝(평가금액/수익률 컬럼 뒤)에 `›` 화살표가 `--muted-foreground` 톤으로 표시된다.
- 평가금액·수익률 숫자와 화살표가 겹치지 않으며 숫자 가독성(tabular-nums, 우측 정렬)이 유지된다.
- 계좌 헤더(summary) 행의 펼침용 `›`(회전)와는 역할이 구분된다 — 잎 행 화살표는 회전하지 않는 정적 진입 표식.

**상호작용 계약**:
- 화살표가 있는 행을 탭하면 해당 종목 상세 시트(`/stocks/[symbol]` → @sheet 인터셉트)가 열린다.
- 양쪽 모드 모두 시트가 스크롤 점프 없이 열린다(`scroll={false}` 통일).

**불변(회귀 금지)**:
- 정렬(평가액/수익률/수익금/이름) 및 계좌별 접이식(`<details>`) 동작 불변.
- 행 전체가 탭 타깃이라는 점 불변(화살표만 누르게 좁히지 않는다).
- 긴 종목명은 말줄임(…) 처리되어 화살표 영역을 침범하지 않는다.

## 수용 검증(시각)

| 계약 | 확인 방법 |
|------|-----------|
| C1 | 대시보드 홈에서 좌상단 ENUF 워드마크 확인, 360px·480px 폭에서 겹침 없음 |
| C2 | 보유 카드의 각 행 우측 `›` 확인, 행 탭→상세 시트 오픈, 정렬/접이식 회귀 없음 |
