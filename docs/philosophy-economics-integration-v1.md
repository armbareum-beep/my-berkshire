# 정치철학·경제사상·철학 통합 정리 (v1)

> 시장을 맹신하지 말고, 정부도 맹신하지 말고, 이론도 맹신하지 말며,
> 항상 반증 가능성을 열어둔 채 확률적으로 행동하는 것이 가장 현실적인 사고방식이다.

이 문서는 my-berkshire(합리적 자본) 프로젝트의 사상적 배경 노트다.
경제사상의 흐름(스미스 → 케인즈 → 하이에크·프리드먼 → 폴라니)과
철학의 흐름(모더니즘 → 포스트모더니즘 → AI 시대)을 정리하고,
이를 투자 태도(실용주의 + 반증주의 = 버핏)와 연결한다.

---

## 1. 경제사상의 흐름: 자유방임주의 → 케인즈 → 신자유주의 → 폴라니

### ① 고전 자유방임주의 (애덤 스미스)

- **핵심**: 시장은 스스로 균형을 찾는다. 정부는 최소한만 개입한다. — "보이지 않는 손"
- **정부 역할**: 국방, 치안, 사법
- **한계**: 대공황으로 시장 실패 발생

### ② 케인즈주의

- **등장 배경**: 1929년 대공황
- **핵심**: 평소에는 시장경제 유지, 시장이 실패하면 정부 개입
- **정부 역할**: 재정지출, 금리정책, 경기부양, 실업 해결
- **대표 사례**: 뉴딜 정책
- **목표**: 자본주의를 살리는 것

### ③ 하이에크 · 프리드먼 (신자유주의)

- **등장 배경**: 1970년대 스태그플레이션
- **핵심**: 정부가 너무 커지면 자유가 사라진다
- **정부 역할**: 법치, 재산권 보호, 최소한의 규제, (프리드먼) 통화량 관리
- **목표**: 자유시장 회복

### ④ 폴라니

- **핵심 문장**: "시장은 사회 안에 있어야 한다."
- **뜻**: 시장 자체를 없애자는 것이 아니다. 노동·토지·화폐는 단순한 상품이
  아니므로 사회가 보호해야 한다.
- **대표 제도**: 노동법, 사회보험, 환경규제, 금융규제
- **케인즈와 차이**:
  - 케인즈 → 경기 안정
  - 폴라니 → 시장의 경계 설정

---

## 2. 우파적 포퓰리즘

- **대표**: 카르데나스, 바르가스, 페론
- **특징**: 노동자 보호, 복지 확대, 농지개혁
- **하지만**: 자본주의 유지, 사유재산 인정, 사회주의 반대

즉, **분배는 하지만 혁명은 하지 않는다.**

---

## 3. 헤게모니 (그람시)

사람은 강제로만 지배당하지 않는다.
사람들이 **"이 사회가 당연하다"**라고 믿게 만드는 힘이다.

| 지배 방식 | 수단 |
| --- | --- |
| 강제 | 경찰, 군대, 법 |
| 동의 | 학교, 언론, 종교, 문화 |

**헤게모니 = 자발적 동의에 의한 지배**

---

## 4. 모더니즘

- **핵심**: 이성, 과학, 진보, 객관성, 새로운 것
- **믿음**: 인간은 이성으로 세상을 설명할 수 있다

---

## 5. 포스트모더니즘

- **핵심**: 거대한 정답은 없다
- **특징**: 다양성, 여러 관점, 권위 의심, 절대 진리 의심
- **거대서사 비판**: "자유시장이 답이다"도, "사회주의가 답이다"도 — 둘 다 의심한다

---

## 6. 실용주의

- **대표**: 퍼스, 윌리엄 제임스, 존 듀이
- **핵심**: 진리보다 **"잘 작동하는가?"**가 중요하다

---

## 7. 반증주의

- **대표**: 칼 포퍼
- **핵심**: 과학은 증명이 아니라 **반증**으로 발전한다
- 즉, 항상 **"내가 틀릴 수도 있다"**를 전제로 한다

---

## 8. 철학 비교

| 철학 | 태도 |
| --- | --- |
| 이성주의 | 진리를 찾자 |
| 실용주의 | 잘 작동하면 된다 |
| 반증주의 | 언제든 틀릴 수 있으니 계속 검증하자 |
| 포스트모더니즘 | 애초에 절대 진리가 있는지도 의심하자 |

---

## 9. 투자와 연결

### 모더니즘적 투자

- 거시경제, 퀀트, 경제모형, 하나의 원리
- **장점**: 확신, 일관성
- **단점**: 크게 틀릴 위험

### 포스트모더니즘적 태도

- 하나의 이론을 맹신하지 않는다
- 여러 관점을 본다

### 실용주의

- 돈 벌면 사용, 안되면 버림

### 반증주의

- 예측보다 검증
- 틀리면 수정
- 확률적 사고

### 버핏과 가장 가까운 철학

**실용주의 + 반증주의**

> 예측은 실패한다. 대응이 중요하다.

---

## 10. 시대 흐름

**경제학 흐름**

```
애덤 스미스 → 케인즈 → 하이에크·프리드먼 → 폴라니
```

**철학 흐름**

```
모더니즘 → 포스트모더니즘 → AI 시대 (새로운 모더니즘?)
```

**AI 시대 특징**

- 과학과 데이터 신뢰 증가
- 동시에 진실과 정보는 더욱 불확실
- → 모더니즘과 포스트모더니즘이 공존하는 시대

---

## 11. 기축시대 vs 현대

| | 기축시대 | 현대 |
| --- | --- | --- |
| 주제 | 인간은 어떻게 살아야 하는가? | 사회는 어떻게 작동하는가? |
| 대표 | 공자, 노자, 석가, 소크라테스, 플라톤 | 케인즈, 하이에크, 포퍼, 폴라니, 푸코, 비트겐슈타인 |

**AI 시대**: 다시 "인간이란 무엇인가?"라는 기축시대적 질문으로 돌아가는 중.
